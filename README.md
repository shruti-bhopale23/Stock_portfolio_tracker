# 📊 Stock Portfolio Tracker

  A simple Python program to track stock investments. It allows users to input stock names and quantities, calculates total investment value, and saves the portfolio to a CSV file.

---

# Features

-  Add stocks and quantities
-  Automatically calculates total investment
-  Updates existing stock entries
-  Handles invalid inputs
-  Saves data to `portfolio.csv`

---

#Technologies Used

- Python
- CSV module
